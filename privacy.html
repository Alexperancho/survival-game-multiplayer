Política de Privacidad — Survival Game

Última actualización: 21/08/2025

Esta app de Android es un contenedor (Trusted Web Activity) que abre el sitio survivalgame.fun. Toda la funcionalidad del juego se ejecuta en la web.

Datos que tratamos

  • Nombre/alias que introduces para jugar.
  • Datos técnicos estándar (como dirección IP, logs de servidor) necesarios para operar el multijugador y prevenir abusos.

No pedimos datos sensibles ni acceso al teléfono (contactos, cámara, ubicación, etc.). No mostramos anuncios.

Uso de los datos

  • Permitir partidas multijugador y funcionamiento del juego.
  • Seguridad, diagnóstico de errores y prevención de abusos.

Conservación

Los registros técnicos se conservan durante el tiempo necesario para operar y depurar el servicio.

Terceros

No compartimos datos con terceros con fines publicitarios. El juego usa socket.io para comunicación en tiempo real.

Tus derechos

Puedes solicitar eliminación de datos de logs razonablemente identificables escribiendo a contact@survivalgame.fun.
